類似文書検索の簡易サンプル実装です。


(0) 準備

	検索対象データのサンプルとして、某QAサイトのデータ１万件分を特徴ベクトル化したものを使います。
	(タイトルおよび複数の特徴後とスコアからなるリスト）

(1) build.pl

	sample.txtをもとにインデックスを作ります。
	Luigiの内部では非階層型クラスタリングを行います。
	それらのクラスタの重心ポイントを使ってさらにクラスタリングを繰り返し、最終的に階層型のツリーを構成します。

	その過程でクラスタリングツールBayonを利用しています。なので事前にインストールしておく必要アリです。


(2) search.pl

	上記で生成したインデックスファイルを使って任意のキーワードとの類似度の高いものを検索して表示します。

	入力されたキーワードのベクトル化にはLingua::JA::Expandを利用しています。この部分は都度YahooAPIをたたくので時間がかかります。
	（その後の類似文書の検索処そのものは非常に高速です)

	なお、このサンプルでは類似度が0.1以上のもののみを表示します。また検索対象データが小さいので（１万件）いろいろなキーワードにはヒットしません。
